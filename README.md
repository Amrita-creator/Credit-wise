## Project Overview

CreditWise is a supervised machine learning project that predicts whether a loan application will be approved or rejected based on applicant financial and personal details.
The goal of this project is to demonstrate the use of supervised learning in a real-world financial problem.

## Features

* Data preprocessing and cleaning
* Feature selection and encoding
* Model training using supervised learning
* Model evaluation and prediction

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

## Dataset

The dataset contains information such as:

* Applicant income
* Loan amount
* Credit history
* Employment details
* Loan status (Approved / Not Approved)
